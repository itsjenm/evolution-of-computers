Evolution of Computer Languages

Year Computer Languages Author Description State of use (Today)
1952 Autocode Alick Glennie In 1952 Autocode was developed for the Mark 1 computer at the University of Manchester. It was the first language that could be translated into machine code. COBOL and FORTRAN are the languages referred to as Autocode today
1958 FORTRAN Joh Backus John Backus created FORmula TRANslation or FORTRAN in 1957. Used today in supercomputers and scientific and mathematical computations.
1958 ALGOL and LISP American and European scientists ALGOL stands for ALGOrithmic Language. It became the point of origin for world-renowned programming languages such as Pascal, Java, C, and C++. In the same year, John McCarthy invented the List processor or LISP. Intended for AI, companies like Boeing and Genworks are still using it. ALGOL was mostly used by computer scientists and frequently updated by the 1950s, but now it's rarely used. LISP is still used for AI programming.
1959 COBOL Developed by a team led by Dr. Grace Murray Hppper COBOL stands for Common Business Oriented Language. It was designed for credit card transaction processors, traffic signals, and phone calls. Used today in banking and gaming. Today's virtual PBX system and other modern solutions to communication
1964 BASIC Developed by Dartmouth College students It stands for Beginners All-Purpose Symbolic Instruction Code. BASIC was then further developed and marketed as a product by Bill Gates. A popular version of BASIC today is QBASIC and used in many business applications and is still considered a valid choice as a programming language.
1970 PASCAL Niklaus Wirth Named after Blaise Pascal, a French mathematician, PASCAL is an imperative and procedural programming language intended to encourage good programming practices using structured programming and data structuring. It's an easy to learn languages used by Apple. Today, it's been mostly replaced by C, C++ and Java, but it's still used as an introduction to programming.
1972 C, SQL Dennis Ritchie C is a general-purpose computer programming language. Designed to be used with the Unix operating system. In the same year, Raymond Boyce and Donald Chamberlain developed SQL (Structured Query Language) which was used for viewing and altering information in databases. Widely used for developing system software, application software, and embedded systems in OS kernels, UNIX or Unix-like software.
1983 C++ Bjarne Stroustrup Modified from C, C++ is the fastest high-level programming language. C++ is an object-oriented programming language which gives a clear structure to programs and allows code to be reused, lowering development costs. Used in MS Office, Adobe Photohop, and various types of high-performance software.
1991 Python Guido Van Rossum Python is a high-level general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Today, Python is one of the most popular programming languages worldwide. Giants like Google and Spotify use it. It's commonly used for developing websites and software, task automation, data analysis, and data visualization.
1995 Java, Javascript James Gosling - Java and Brendan Eich - JavaScript Java is one of the world's most famous and popular programming languages. It's a general purpose, high level language used in cell phones and parking machines. JavaScript is used for web development, PDF documents, and desktop widget. It's almost in every major website. Today, Java is used for a variety of things but mostly for building and running mobile applications. JavaScript adds interactivity to websites and used in web-based applications today.
